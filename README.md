# funel-software

Landing และหน้าร้านขาย Digital Product / Funnel สร้างด้วย Next.js

## Getting Started

รันเซิร์ฟเวอร์พัฒนา:

```bash
npm run dev
```

เปิด [http://localhost:3000](http://localhost:3000) ในเบราว์เซอร์

## โครงสร้าง

- `src/app` – หน้าและ layout (App Router)
- `src/components` – คอมโพเนนต์共用 (Navbar, Footer, ProductCard)
- `src/lib` – ข้อมูลสินค้าและ utilities

## Deploy

ใช้ [Vercel](https://vercel.com) หรือ platform ที่รองรับ Next.js
